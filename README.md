# AI Content Factory

## 專案目標

建立一套 AI 自動化內容工廠。

整個流程不需要人工搬運資料。

所有 AI 必須自動接棒。

人只負責：

1. 新增品牌
2. 批准發布

除此之外全部由 AI 完成。
